# 推导 5: AdamW 更新规则

## SGD 基础

标准 SGD：

```
θ_t = θ_{t-1} - lr × g_t
```

g_t = ∂L/∂θ_{t-1} 是当前梯度。

问题：每个参数用相同的学习率，但不同参数的梯度量级可能差几个数量级。

---

## Adam：自适应学习率 + 动量

### 一阶矩（动量）

```
m_t = β₁ × m_{t-1} + (1 - β₁) × g_t
```

指数移动平均，平滑梯度方向。β₁ = 0.9 意味着最近 ~10 步的平均。

### 二阶矩（自适应学习率）

```
v_t = β₂ × v_{t-1} + (1 - β₂) × g_t²
```

梯度的平方的指数移动平均，估计每个参数的梯度方差。β₂ = 0.999 意味着最近 ~1000 步的平均。

### 偏差修正

m_t 和 v_t 初始化为 0，导致初期估计偏小。修正：

```
m̂_t = m_t / (1 - β₁^t)
v̂_t = v_t / (1 - β₂^t)
```

t → ∞ 时，β^t → 0，修正系数 → 1，偏差消失。

**为什么需要修正**：假设 β₁ = 0.9，第一步 m₁ = 0.1g₁，但 m̂₁ = 0.1g₁/0.1 = g₁。如果不修正，前几十步的学习率会偏低。

### Adam 更新

```
θ_t = θ_{t-1} - lr × m̂_t / (√v̂_t + ε)
```

- m̂_t：梯度方向（平滑后）
- √v̂_t + ε：梯度大小的自适应缩放（ε = 1e-8 防止除零）
- 效果：梯度大的参数步长被缩小，梯度小的参数步长被放大 → 所有参数的更新量级一致

---

## Adam + L2 正则（Adam 的 weight decay）

L2 正则：在 loss 中加 λ/2 × ||θ||²，等价于在梯度上加 λθ：

```
g_t = ∂L/∂θ + λ × θ_{t-1}       # 梯度 + L2 正则项
```

然后这个 g_t 进入 Adam 的 m_t 和 v_t：

```
m_t = β₁ m_{t-1} + (1-β₁)(∂L/∂θ + λθ)
v_t = β₂ v_{t-1} + (1-β₂)(∂L/∂θ + λθ)²
```

### 问题：L2 正则被自适应缩放破坏了

更新量中来自 weight decay 的部分：

```
decay contribution ≈ lr × λθ × (自适应缩放因子)
```

其中自适应缩放因子 = m̂_t / (√v̂_t + ε)，对不同参数**不同**：

- 梯度大的参数：√v̂_t 大 → 缩放因子小 → 实际 decay 小
- 梯度小的参数：√v̂_t 小 → 缩放因子大 → 实际 decay 大

**结论**：weight decay 的效果被自适应学习率扭曲了。本意是均匀衰减所有参数，实际上衰减量随梯度大小而变。

---

## AdamW：解耦 Weight Decay

Loshchilov & Hutter (2019) 的解决方案：**把 weight decay 从梯度中拿出来，直接作用于参数**。

```
θ_t = θ_{t-1} - lr × m̂_t / (√v̂_t + ε)  -  lr × λ × θ_{t-1}
       \_________________________/          \______________/
           Adam 自适应更新                     纯粹的 weight decay
```

等价写法：

```
θ_t = (1 - lr × λ) × θ_{t-1} - lr × m̂_t / (√v̂_t + ε)
```

### 两种写法的等价性验证

```
方法 1 (AdamW):
  θ_t = θ_{t-1} - lr × m̂/(√v̂+ε) - lr×λ×θ_{t-1}
      = θ_{t-1}(1 - lr×λ) - lr × m̂/(√v̂+ε)

方法 2 (重排):
  θ_t = (1 - lr×λ) × θ_{t-1} - lr × m̂/(√v̂+ε)  ✓ 相同
```

### AdamW vs Adam+L2 的唯一区别

```
Adam+L2:  g_t ← g_t + λθ          # decay 在梯度里，进入 m/v
AdamW:    θ ← θ - lr×λθ           # decay 直接作用参数，不进入 m/v
```

---

## 详细对比

### 第 t 步的完整更新

**Adam + L2**：

```
g_t = ∂L/∂θ + λθ
m_t = β₁ m_{t-1} + (1-β₁) g_t
v_t = β₂ v_{t-1} + (1-β₂) g_t²
θ_t = θ_{t-1} - lr × m̂_t / (√v̂_t + ε)
```

decay 项 λθ 经过 m_t 和 v_t 的非线性变换。

**AdamW**：

```
g_t = ∂L/∂θ                      # 纯梯度，不含 decay
m_t = β₁ m_{t-1} + (1-β₁) g_t
v_t = β₂ v_{t-1} + (1-β₂) g_t²
θ_t = θ_{t-1} - lr × m̂_t / (√v̂_t + ε) - lr × λ × θ_{t-1}
```

decay 项直接作用于 θ，不经过 m/v。

### 数值示例

假设某参数 θ = 1.0，lr = 0.001，λ = 0.01，m̂/(√v̂+ε) = 0.1：

```
Adam+L2 (假设 √v̂ 缩放因子 = 0.5):
  effective_decay = lr × λ × θ × 缩放因子 = 0.001 × 0.01 × 1.0 × 0.5 = 0.000005

AdamW:
  decay = lr × λ × θ = 0.001 × 0.01 × 1.0 = 0.00001  (恰好 2 倍)

另一个参数 (缩放因子 = 2.0):
  Adam+L2: 0.001 × 0.01 × 1.0 × 2.0 = 0.00002  (4 倍于第一个参数!)
  AdamW:   0.001 × 0.01 × 1.0 = 0.00001        (与第一个参数相同)
```

## 为什么 LLM 训练用 AdamW

1. **参数量大**（数十亿），不同层的梯度量级差异巨大
   - Embedding 层：梯度小但参数多
   - Attention 层：梯度大小取决于 attention pattern
   - FFN 层：梯度通常最大

2. **自适应学习率是必须的**：一个统一的学习率无法同时满足所有层的需求

3. **解耦 weight decay 是必须的**：
   - Adam+L2：大梯度层的 decay 被压缩，小梯度层的 decay 被放大 → 正则化不均匀
   - AdamW：所有参数的 decay 比例 = lr × λ，均匀一致

4. **实验验证**：Loshchilov & Hutter 的论文中，AdamW 在图像分类任务上显著优于 Adam+L2。LLM 社区普遍采用 AdamW 作为标准优化器。

---

## 常用超参

```
β₁ = 0.9         # 动量衰减（最近 ~10 步）
β₂ = 0.999       # 方差衰减（最近 ~1000 步）
ε = 1e-8          # 防除零
λ = 0.01          # weight decay
lr = 2e-4 ~ 1e-5  # 学习率（LoRA 常用 2e-4）
```

---

## 关键理解

| 问题 | 答案 |
|---|---|
| AdamW vs Adam+L2 的区别？ | Weight decay 不经过 m/v，直接作用于参数 |
| 为什么解耦更好？ | 所有参数的 decay 比例一致，不被自适应缩放扭曲 |
| 为什么 LLM 必须用 AdamW？ | 不同层梯度量级差几个数量级，L2 正则会不均匀 |
| AdamW 的唯一修改？ | 把 `-lr×λ×θ` 从梯度项移到参数更新项 |
